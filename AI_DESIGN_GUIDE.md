## üéØ Recommended: Uizard AI (Free & Easy)

### Step 1: Sign Up
1. Go to **uizard.io**
2. Click "Sign up free"
3. Choose "Mobile App" template

### Step 2: Generate Each Screen with AI

#### Screen 1: Home Screen
**Prompt:**
```
Emergency medical assistance app home screen for elderly users. 
Large red circular SOS button (280px) in the center with white 
SOS text. App title "SafeRide" at top. Subtitle "Emergency Help 
Available 24/7". Four quick action cards below in 2x2 grid: 
My Profile, My Location, Contacts, Settings. Each card has icon 
and label. Green banner at bottom saying "You're Protected". 
Clean white background, high contrast, large text for readability.
```

#### Screen 2: Emergency Active
**Prompt:**
```
Emergency alert screen with red gradient header. Large alert 
icon and "HELP REQUESTED" title in white. Text "Emergency 
services notified" below. Blue card showing location shared 
with address "742 Evergreen Terrace". Information checklist 
with green checkmarks for medical conditions, medications, 
and emergency contacts sent. Green "Call Dispatcher" button 
and gray "Cancel Emergency" button. Teal banner at bottom: 
"Help is on the way".
```

#### Screen 3: Tracking Screen
**Prompt:**
```
Ambulance tracking screen with teal gradient banner showing 
"6 min" in large text and "Estimated Arrival" label. Map 
placeholder area with ambulance emoji icon, text "Live Tracking 
Active" and "2.3 miles away". Timeline section titled "Status 
Updates" with four items: ambulance dispatched, caregiver 
notified, location shared, medical info sent. Each with icon 
and timestamp. Green "Call Ambulance" button at bottom.
```

#### Screen 4: User Profile
**Prompt:**
```
User profile screen with purple gradient header. Elderly man 
avatar emoji, name "Robert Thompson", age "72 years old" in 
white text. Medical information card showing blood type O+, 
conditions: diabetes and high BP, allergies: penicillin, 
medications listed. Home address card. Orange highlighted 
emergency contact card with daughter's name Sarah Thompson 
and phone number. Purple "Edit Profile" button at bottom.
```

#### Screen 5: Caregiver Dashboard
**Prompt:**
```
Caregiver dashboard screen with title at top. Red bordered 
emergency alert card with "Active Emergency" header, patient 
info "Robert Thompson (Father)" with avatar, location display, 
ambulance response status with green checkmark. Two buttons: 
"Call" and "Track". Health summary card below. Notification 
settings card with toggle switches for emergency alerts, 
daily reports, medication reminders.
```

#### Screen 6: Settings Screen
**Prompt:**
```
Settings screen with back button and "Settings" title. Four 
grouped card sections. First: notification toggles for 
Notifications, Voice Commands, Vibration, Voice Announcements 
with green toggle switches. Second: edit profile, medical info, 
emergency contacts, location services with right arrows. 
Third: large text mode, high contrast, accessibility options. 
Fourth: help & support, privacy policy, about SafeRide.
```

### Step 3: Customize Each Screen
1. Adjust colors to match our palette:
   - Red: #E53E3E
   - Purple: #667EEA
   - Teal: #38B2AC
   - Green: #48BB78
2. Increase font sizes (16px minimum)
3. Add proper spacing (15-30px)

### Step 4: Export to Figma
1. Click "Export" in Uizard
2. Choose "Figma" format
3. Import to your Figma account

## üé® Alternative: Galileo AI

### Step 1: Sign Up
- Go to **usegalileo.ai**
- Sign up (free trial available)

### Step 2: Use Single Comprehensive Prompt
```
Design a mobile app called SafeRide for emergency medical 
assistance for elderly and disabled people. Include these 
6 screens:

1. Home screen with large red SOS emergency button (280px 
circle), app title, 4 quick action cards, protection status

2. Emergency active screen with red alert header, location 
shared confirmation, medical info checklist, call and cancel 
buttons

3. Tracking screen with ETA banner, map with ambulance location, 
status timeline, call button

4. User profile with purple header, medical information (blood 
type, conditions, allergies, medications), address, emergency 
contact

5. Caregiver dashboard with active emergency alert, patient 
info, health summary, notification toggles

6. Settings screen with grouped settings cards, toggles for 
notifications and accessibility features

Design must be accessible with large text (16px+), high 
contrast colors, simple navigation, suitable for elderly 
users with limited tech skills.
```

### Step 3: Review & Edit
- Galileo generates all screens at once
- Edit individual elements as needed
- Export to Figma
